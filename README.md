# Aerospace Data Analytics Portfolio
This repository features professional Python tools for processing flight test data and visualizing aircraft performance metrics.

## 🚀 Projects Overview
### 1. Flight Data ETL Pipeline
- **Description:** An automated system for Extracting, Transforming, and Loading (ETL) raw sensor data.
- **Key Features:** Automated data cleaning via linear interpolation and structural load flagging.
- **Verification:** Includes unit testing (`pytest`) to ensure mathematical integrity.

### 2. Interactive Aero-Performance Dashboard
- **Description:** A web-based tool for aerodynamic design studies.
- **Key Features:** Dynamic drag polar generation and real-time Lift-to-Drag (L/D) optimization.

## 🛠 Tech Stack
- **Languages:** Python
- **Libraries:** Pandas, NumPy, Plotly, Streamlit
- **Quality Assurance:** Pytest (Unit Testing)
